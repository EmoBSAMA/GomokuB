# GomokuB
五子棋游戏，B佬君提供<br>
开坑于2018年6月18日，但是实际上真正开坑日期是2019年7月25日。这天我在另外一个project上面上传了我第一份具有实际意义的小工具（双人模式的圈叉棋）。因此现在有一定能力去做此项目了。开坑开坑。<br>

此项目目前只能在Python IDLE上正常运行，目测在GUI化后能兼容性提高<br>
